# Troubleshooting

`Malformed pairing` means names must end in `_Red.idat[.gz]` and `_Grn.idat[.gz]` with the same preceding path/basename. A rank-deficient design requires removing confounded covariates or changing the formula. DMP analysis requires a contrast and sufficient replication. DMR calls stop rather than use unverified coordinates if the EPICv2 DMRcate manifest/annotation packages are absent.
